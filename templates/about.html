About LibPortal

LibPortal is a modern, responsive full-stack platform designed to facilitate catalog operations and book issue-return tracking for public libraries.

It includes detailed modules for staff administrators, members databases, automated fine calculations, overdue alerting engines, and visual dashboard reporting using advanced charting.

Key Technology Highlights

  • Backend Framework: Django 5.x with SQLite for transactional relational data storage.
  • Design System: Bootstrap 5 with responsive CSS variables, layout transitions, glassmorphic panels, and gradient headers.
  • Client Scripting: Vanilla ES6 Javascript featuring debounced AJAX search hooks, localStorage selectors, and live calculations.
  • Asset Delivery: WhiteNoise engine supporting brotli compression and caching, configured for Render deployment.
